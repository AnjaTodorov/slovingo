<h1 align="center"> Slovingo </h1>

## Description
Slovingo is a mobile application for learning the Slovenian language through interactive lessons, quizzes, daily vocabulary, translation, and AI-assisted conversation.  
The application follows a structured learning program with progressive difficulty levels and tracks user progress locally using a relational database.



## Vision
The vision of Slovingo is to provide users with an accessible and intuitive tool for learning Slovenian, regardless of their starting knowledge level.  
By combining structured lessons, intelligent feedback, and AI assistance, the application aims to make language learning efficient and motivating.


## Functionalities
- Creating a structured study program (learning levels)
- Solving lessons and quizzes with different difficulty levels 
- Automatic tracking of learning progress
- Daily vocabulary feature (“Word of the Day”)
- English ↔ Slovenian translation
- AI-powered chat tutor for language assistance
- User profile with progress overview

## Installation & Execution

### Preconditions
- Flutter SDK  
- Android Studio with Android SDK & Emulator  
- Visual Studio Code (recommended)  
- Git  

Verify Flutter installation:
```bash
flutter doctor
```

---

### Installation (Namestitev)
```bash
git clone https://github.com/AnjaTodorov/slovingo.git
cd slovingo
flutter pub get
```

---

### Execution
```bash
flutter run
```
